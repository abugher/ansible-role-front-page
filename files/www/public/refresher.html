What is this? This page keeps login sessions from expiring by continually refreshing authenticated connections. Add a URL to create a new entry in the list. Remove an entry to stop refreshing that page. When you have added enough sites, bookmark this page to preserve your list of pages to be prepared in case of browser session loss.

Why doesn't it work? It works for some sites. Most sites, though, will set X-Frame-Options to sameorigin or deny. In that case, your browser will refuse to load the site in a frame. (It may also claim that the site refused or did not allow the connection, which is a blatant lie.)

Can I make it work? You can find a browser extension to modify your browser's policies about this header, but it would compromise your security to ignore it in general. Instead, copy the source code of this page (refresher.html and refresher.js) and browse the copy. Apply any policy overrides only to your offline copy of this page.
